# Examples

This directory contains example scripts demonstrating various features of LiteQ.

## Running Examples

```bash
# Basic usage
python examples/basic.py

# Multiple queues
python examples/multiple_queues.py

# Priority-based execution
python examples/priorities.py

# Monitoring and management
python examples/monitoring.py

# Real-world scenario: Email campaign
python examples/email_campaign.py
```

## Examples Overview

- **basic.py** - Simple introduction to LiteQ with async and sync tasks
- **multiple_queues.py** - Using multiple named queues with different workers
- **priorities.py** - Demonstrates task priority execution order
- **monitoring.py** - Queue monitoring, statistics, and task management
- **email_campaign.py** - Real-world email campaign system example

Press `Ctrl+C` to gracefully stop any running example.
